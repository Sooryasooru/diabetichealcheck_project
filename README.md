# Diabetes Clustering Project

## Overview
This project performs clustering analysis on a diabetes dataset.

## Features
- Data Cleaning
- Feature Engineering
- Feature Scaling
- PCA
- KMeans Clustering
- EDA Visualization
- Model Evaluation
- Pytest Testing
- Pylint Code Quality
- GitHub Actions CI/CD

## Run Project

```bash
pip install -r requirements.txt
python main.py
```

## Run Testing

```bash
pytest
```

## Run Pylint

```bash
pylint src/
```
